# Node.js Example

Minimal Node.js setup for running the [`minecraft-smp`](https://www.npmjs.com/package/minecraft-smp) package.

## Usage

Install dependencies:

```sh
pnpm install
```

Build TypeScript sources:

```sh
pnpm build
```

Run the app:

```sh
pnpm start
```

## Notes

- Depends on local `minecraft-smp` package via workspace.
